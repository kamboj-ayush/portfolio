# Asset Placeholders

This directory contains placeholder instructions for your portfolio assets.

## Required Files

### Profile Image
- **Path**: `/public/profile.jpg`
- **Recommended size**: 800x800px (square)
- **Format**: JPG, PNG, or WebP
- **Description**: Your professional profile photo

### Resume
- **Path**: `/public/resume.pdf`
- **Format**: PDF
- **Description**: Your resume/CV file

### Open Graph Image
- **Path**: `/public/og-image.jpg`
- **Recommended size**: 1200x630px
- **Format**: JPG or PNG
- **Description**: Image for social media previews

## Project Assets

Add your project screenshots to:
- `/public/assets/projects/`

### Naming Convention
- Thumbnails: `project-name-thumb.jpg`
- Screenshots: `project-name-1.jpg`, `project-name-2.jpg`, etc.
- Recommended size: 1200x630px for thumbnails

## Quick Setup

1. Add your profile photo as `profile.jpg`
2. Add your resume as `resume.pdf`
3. Add project images to `assets/projects/`
4. Update paths in `data/profile.json`

That's it! No code changes needed.
